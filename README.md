#粉丝团经济
https://www.ximalaya.com/jiaoyu/32763720/

《粉丝团经济》这本书凝结了肖威2014~2020年的人生经历和创业经历的总和，

在这本书里面主要去完成了三个知识体系的梳理，其中包含的是财务的知识，包含的是产品的知识，包含的是销售的知识。

通过阅读这本书，能够非常快速的获得在职场环境当中立足的能力，从而逐渐的具备创业化的思维，打造自己的企业，从而逐渐的获得时间支配的能力。
